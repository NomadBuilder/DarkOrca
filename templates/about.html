About DarkOrca

DarkOrca is a comprehensive security assessment platform designed for defensive security reconnaissance and risk scoring. It provides automated scanning capabilities to identify vulnerabilities, misconfigurations, and security weaknesses across web applications and services.

Built with security professionals in mind, DarkOrca combines both passive reconnaissance and active exploitation testing to deliver thorough security assessments with detailed risk analysis and actionable remediation guidance.

Key Features

  • Comprehensive Scanning - Test for 30+ vulnerability types including XSS, SQL Injection, SSRF, XXE, command injection, and more
  • Multiple Scan Modes - Choose from Defensive (passive), Offensive (active exploitation), or Comprehensive (both) scanning modes
  • Risk Scoring - Automated risk assessment with weighted scoring (0-100) and categorized risk levels (Critical, High, Medium, Low, Minimal)
  • Evidence Collection - Detailed evidence capture including HTTP requests/responses, payloads, and exploitation details
  • WordPress Security - Specialized scanners for WordPress installations including plugin vulnerabilities, theme analysis, and user enumeration
  • API Security - GraphQL endpoint testing, JWT security analysis, and REST API security assessments
  • Modern Web Technologies - Support for WebSocket security, template injection (SSTI), and deserialization vulnerability testing
  • Professional Reports - Export findings to JSON, Markdown, or PDF with optional AI-generated executive analysis
  • Email Notifications - Get notified when scans complete via email with summary and direct links to results
  • Shareable Results - Share scan results via unique URLs with 30-day expiration

Defensive Mode

Passive, read-only scanning that performs reconnaissance and discovery without attempting exploitation. Safe for production systems, perfect for initial assessments and compliance checks.

Offensive Mode

Active scanning with exploitation attempts to confirm vulnerabilities. Should only be used with explicit authorization as it may impact system stability or trigger security alerts.

Comprehensive Mode

Combined approach that performs both defensive reconnaissance and offensive exploitation testing. Provides the most thorough security assessment available.

Usage Guidelines

Authorization Required

Only use DarkOrca on systems you own or have explicit written authorization to test. Unauthorized scanning is illegal and may result in criminal charges. Always ensure you have proper authorization before performing security assessments, especially in Offensive or Comprehensive modes.

Production Safety

Defensive mode is designed to be safe for production environments as it performs only passive reconnaissance. However, Offensive mode can impact system stability, trigger security alerts, and potentially cause service disruption. Always test in staging environments first when possible.

Best Practices

Review all findings carefully, as false positives can occur. Use evidence provided with each finding to verify vulnerabilities. Prioritize critical and high-severity findings, but don't ignore lower-severity issues that could combine to create larger security risks. Regularly scan your applications to catch new vulnerabilities introduced during development.
